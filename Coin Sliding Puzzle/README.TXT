------------------------------------------------------------------------
This is a GUI-based application that allows a user to solve the
puzzle by interactively moving (“sliding”) coins displayed in a frame. 
When the application begins, a frame displays the arrangement of numbered 
coins. A user can slide a coin to a new position by dragging the coin with 
the mouse. The GUI have a Reset button that restores the coins to 
the original confi gurationand also an Exit button.
