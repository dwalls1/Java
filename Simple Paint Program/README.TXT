------------------------------------------------------------------------
This is a drawing application that allows a user to pencil-draw figures
by dragging the mouse. Drawing can be done using colors chosen via an 
input dialog box. The application also provides an Erase button that 
clears the screen.
------------------------------------------------------------------------

VERSION or DATE: 2018
